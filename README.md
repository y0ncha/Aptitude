# Aptitude

Central documentation hub for the **Aptitude ecosystem**.

This repository consolidates:
- Documentation related to Aptitude
- Links to all Aptitude code repositories
- Session notes and summaries

## Repositories

- **[aptitude-server](https://github.com/y0ncha/aptitude-server)** – Repository and registry service for Aptitude assets.

- **[aptitude-client](https://github.com/y0ncha/aptitude-client)** – Client and resolver responsible for dependency resolution and asset retrieval.
